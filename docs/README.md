# Dokumentacja projektu

Projekt: **B OUT 15 - Ubezpieczenia komunikacyjne**

Ten katalog zawiera dokumentację planistyczną projektu:

- [PRD.md](PRD.md) - wymagania produktu, zakres, architektura, aplikacja demonstracyjna i kryteria sukcesu,
- [DATABASE_DESIGN.md](DATABASE_DESIGN.md) - projekt schematów, tabel, relacji, ról i danych demonstracyjnych,
- [IMPLEMENTATION_PLAN.md](IMPLEMENTATION_PLAN.md) - szczegółowy plan realizacji, zależności, rezultaty i kryteria akceptacji.

Dokumenty są napisane tak, aby stanowiły:

1. podstawę realizacji technicznej,
2. listę zadań możliwą do podziału między członków grupy,
3. mapę pokrycia wszystkich kryteriów oceny,
4. punkt wyjścia do końcowego sprawozdania i siedmiominutowej prezentacji.
